# 📝 Go CLI To-Do App

A simple command-line To-Do application built using Go, with persistent storage using JSON.

## 🚀 Features

* Add tasks
* List tasks
* Mark tasks as done
* Delete tasks
* Persistent storage using JSON file

## 🛠️ Tech Stack

* Go (Golang)
* JSON for storage

## ▶️ Run Locally

```bash
go run main.go
```

## 📦 Future Improvements

* Convert to REST API
* Add database support (MongoDB/MySQL)
* Dockerize the application

## 💡 Learning Outcome

This project helped in understanding:

* Structs and slices in Go
* File handling
* JSON encoding/decoding
* CLI-based interaction
